Souschef shows nutrition information right inside your recipe so you can plan meals with a clear picture of what you're eating. When the source website already includes nutrition, Souschef uses that; when it doesn't, you can generate estimates with AI in a single tap. You can filter your library by calories per serving and edit any value yourself.

## What Souschef shows

Every recipe with nutrition data displays a per-serving breakdown of:

- **Calories**
- **Protein**
- **Carbohydrates**
- **Fat**
- **Additional nutrients** (fiber, sugar, sodium, and more when available)

</Info>

## Where nutrition comes from

Souschef pulls nutrition data from two places:

- **From the source website**, when a recipe is imported from a site that publishes structured nutrition data.
- **From AI estimates**, when the source doesn't include nutrition or you've created a recipe yourself.

AI estimates are **clearly labeled** so you always know whether a number came from the original recipe or was estimated.

## Generate Nutrition with AI

If a recipe doesn't have nutrition information, generate an estimate in one tap.

<Steps>
  <Step title="Open the recipe">
    Tap any recipe in your library.
  </Step>
  <Step title="Tap 'Generate Nutrition'">
    In the nutrition section, tap **Generate Nutrition**. Souschef reads the ingredients, quantities, and servings and produces an estimate for calories, protein, carbs, fat, and additional nutrients.

    <Info>
      📸 Screenshot needed: the 'Generate Nutrition' button on a recipe that has no nutrition data yet.
    </Info>
  </Step>
  <Step title="Review the estimated values">
    The estimated nutrition is labeled so it's clear the values came from AI. Tap any number to edit it if you'd like a more precise value.

    <Info>
      📸 Screenshot needed: the generated nutrition panel with an 'AI estimate' label visible.
    </Info>
  </Step>
</Steps>

<Note>
  AI nutrition estimates are meant as a helpful guide, not medical or dietary advice. Actual values depend on brands, exact quantities, and how you cook — always double-check if you have specific dietary needs.
</Note>

## Edit nutrition manually

You can always override any value. Tap the nutrition section, then tap the number you'd like to edit (calories, protein, carbs, fat, etc.). This is handy when you know a value more precisely than the source or the AI does.

## Filter your library by calories

Looking for something lighter tonight? Use the calorie filter on your library to show only recipes under a certain number of calories per serving.

<Info>
  📸 Screenshot needed: the library filter drawer with the 'calories per serving' filter active.
</Info>

<Tip>
  Nutrition information becomes more useful the more accurate your servings are. Before generating nutrition, double-check the recipe's servings so per-serving values are meaningful.
</Tip>
