Every time you import a recipe, Souschef's AI reads the recipe and suggests a set of tags that describe it — cuisine, meal type, diet, main ingredient, cooking method, and more. Suggested tags are pre-selected but always editable, and they power the filters throughout your library.

## What Souschef tags

Suggested tags cover the ways you actually think about recipes:

- **Cuisine** — Italian, Japanese, Mexican, Thai, and so on
- **Meal type** — breakfast, lunch, dinner, snack, dessert
- **Diet** — vegetarian, vegan, gluten-free, high-protein, and more
- **Main ingredient** — chicken, pasta, tofu, salmon, and so on
- **Cooking method** — baked, grilled, air fryer, one-pot, slow cooker, no-cook
- **More** — quick, kid-friendly, meal prep, holiday, and other contextual tags

## Tags during import

When you import a recipe from any source — [Instagram](/importing/import-from-instagram), [TikTok](/importing/import-from-tiktok), [YouTube](/importing/import-from-youtube), [Pinterest](/importing/import-from-pinterest), [Facebook](/importing/import-from-facebook), a [website](/importing/import-from-website), [photos](/importing/import-from-photos), or [plain text](/importing/import-from-plain-text) — the import preview shows Souschef's suggested tags with the most relevant ones already selected.

You can:

- **Accept the suggestions** as-is with a single tap on Save.
- **Untick tags** that don't feel right.
- **Tap to add** additional tags Souschef didn't suggest.

## Editing tags on a saved recipe

Already saved a recipe? You can still edit its tags anytime.

<Steps>
  <Step title="Open the recipe and tap Edit">
    Open the recipe from your library and tap the **Edit** button.
  </Step>
  <Step title="Update the tags">
    Add or remove tags across any category. Saved changes take effect immediately in your library filters.

    <Info>
      📸 Screenshot needed: the recipe editor with the tags section open and tags selected across cuisine, meal type, and diet categories.
    </Info>
  </Step>
</Steps>

## Filtering your library by tag

Tags become powerful the moment you have more than a handful of recipes. In your library:

<Steps>
  <Step title="Tap the filter icon">
    Open the filters drawer above your recipe list.

    <Info>
      📸 Screenshot needed: the library view with the filters drawer open, showing tag categories.
    </Info>
  </Step>
  <Step title="Pick one or more tags">
    Filter by any combination of cuisine, meal type, diet, main ingredient, cooking method, or more.
  </Step>
  <Step title="See only matching recipes">
    Your library instantly narrows to recipes that match all selected tags.
  </Step>
</Steps>

<Tip>
  Combine a diet tag with a meal type — like **vegetarian** \+ **dinner** — to pull up an instant weeknight shortlist.
</Tip>
